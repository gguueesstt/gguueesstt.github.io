The pattern organ is a camera-based digital musical instrument, that uses a relatively direct mapping between camera data and audio wavetable. The simplicity of these mappings enables the audification of camera data in a way that responds closely to material behaviours playing out in front of the lens.

More information on the pattern organ and its design can be found in these research papers:

  • Entangling with Light and Shadow, NIME'25, Utrecht, Netherlands
  • Harmonic Materials, AM.ICAD'25, Braga, Portugal

The Pure Data patch (which is open source), can be found on github
